## Running the app

```bash
# to install
$ yarn install
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
## Docker
```bash
$ docker-compose up -d

# docker shell
$ docker exec -it mongodb /bin/bash
$ mongo mongodb://localhost:27017 -u admin -p admin
```
